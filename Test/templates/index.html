    模板标签：

    if...else...endif 标签

    {% if userForIF%}
  • name:{{userForIF.name}}
  • age:{{userForIF.age}}
  • sex:{{userForIF.sex}}
    • {% else%} 用户不存在！ {%endif %}

    for 标签

    {%for book in book_list%}
  • {{forloop.counter}}.{{book}}
    • {%endfor%}   {%for k,v in user.items%}
  • {{forloop.counter}}.{{k}}:{{v}}
    • {%empty%} 如果集合为空，则执行empty标签 {%endfor%}
